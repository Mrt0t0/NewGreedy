📋 Log fields
Field / FlagDescription
[SEED] / [DOWN]Torrent mode (pure seeder or downloader)
DL:Cumulative download tracked by the engine
UL:Cumulative upload reported to tracker
+ΔULUpload increment added at this announce
R:Current ratio (UL / DL)
ETA:~XaEstimated announces to reach target ratio
#NTotal announce count for this torrent
[STAG]Deliberate stagnation cycle — no upload added
[STALL_NET]Network stall — N consecutive announces with DL=0
[STALL_ALGO]Algorithm stagnation active
[TARGET_REACHED]Target ratio reached — injection stopped
[PURGED]Torrent removed from stats after event=stopped
